## 🛠 Features

The tool includes several modules to assist with gameplay.

### ESP and Wallhack
This feature draws boxes around enemies. It shows their distance and health status. You can see these indicators through walls to know enemy locations. Adjust the colors and boxes in the settings menu.

### Aimbot
The aimbot adjust your crosshair to track enemy movement. It lock onto targets to land shots. You can change the smoothness and targeting bone in the configuration tab. Use this to maintain focus during firefights.

### Recoil Control
The recoil module manages your weapon sway. It counteracts the camera movement caused by firing to keep your shots straight. This tool works with most automatic weapons currently in the game.

### Triggerbot
The triggerbot fires your weapon when your crosshair detects an enemy. It removes the delay between seeing a target and shooting. Accuracy improves when you use this in combination with the aimbot.

## 🛡 Security and Safety

Developers designed this software to operate as a background process. To keep your system stable, disable other overlays like Discord, GeForce Experience, or Steam overlays while using this tool. These extra overlays might conflict with the draw function and cause crashes. Always keep your Windows OS updated to the latest security patch.

## 🔍 Troubleshooting

If you encounter issues, look through these common solutions.

*   **Tool does not display:** Ensure you run the game in Borderless Windowed mode. Exclusive fullscreen mode blocks external overlays.
*   **Application crashes:** Verify that you launched the tool with administrator rights. 
*   **Aimbot feels shaky:** Lower the "Smoothing" parameter in the settings menu. High values cause stuttering, while low values provide a crisp feel.
*   **FPS drops:** Reduce the update rate of the ESP feature in the configuration panel to save system resources.
